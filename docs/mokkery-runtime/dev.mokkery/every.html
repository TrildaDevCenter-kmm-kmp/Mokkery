every

fun <T> every(block: ArgMatchersScope.() -> T): BlockingAnsweringScope<T>

Starts defining an answer for a single function call specified by block. Returned scope allows defining answer.

Provided block must be a lambda and single mock call must occur directly inside it. Extracting block content to functions is prohibited.
